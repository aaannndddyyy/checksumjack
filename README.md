checksumJack
============

MD5, SHA-1, Ripemd, SHA-2 (SHA254, SHA256, SHA384, SHA512) and Whirlpool.

## INSTALL

libraries needed:
QT5
openSSL -> libSSL-dev

# LINUX 

buildtools needed:
make
gcc or clang

```bash
cd src
qmake -o Makefile checksumJack.pro
make
make install
```

# WINDOWS

Get installer from http://www.checksumjack.org
